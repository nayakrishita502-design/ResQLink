# ResQLink – Smart Emergency Response System

## 🚨 Problem
During emergencies, lack of real-time coordination between victims, responders, and nearby helpers leads to delayed response times and increased risk to lives.

## 💡 Solution
ResQLink enables instant SOS alerts, real-time location sharing, and coordination with nearby volunteers and emergency services.

## 🔥 Features
- One-tap SOS alert
- Live location tracking
- Nearby volunteer notification
- SMS fallback for low internet
- Emergency services integration

## 🛠️ Tech Stack
- Flutter / Web
- Firebase
- Google Maps API
- SMS API
